# 英语单词学习进度追踪

7岁儿童两个月掌握400-500个核心英语单词的学习计划和进度追踪工具。

## 📱 在线访问

访问地址：`https://你的用户名.github.io/spelling/`

## ✨ 功能特点

- ✅ 完整的8周学习计划
- ✅ 交互式进度追踪
- ✅ 单词掌握状态标记
- ✅ 学习笔记记录
- ✅ 数据本地保存
- ✅ 完美适配手机端

## 🚀 部署到GitHub Pages

### 步骤1：创建GitHub账号
1. 访问 https://github.com
2. 点击 Sign up 注册账号

### 步骤2：创建仓库
1. 登录后点击右上角 "+" → "New repository"
2. Repository name 填写：`spelling`
3. 选择 "Public"
4. 点击 "Create repository"

### 步骤3：上传文件
1. 在仓库页面点击 "uploading an existing file"
2. 拖拽以下文件到页面：
   - `index.html`
   - `学习进度追踪.html`
3. 点击 "Commit changes"

### 步骤4：启用GitHub Pages
1. 点击仓库的 "Settings"
2. 左侧菜单找到 "Pages"
3. Source 选择 "main" 分支
4. 点击 "Save"
5. 等待1-2分钟，页面会显示访问地址

### 步骤5：访问网站
- 访问地址：`https://你的用户名.github.io/spelling/`
- 把这个网址发到微信，手机直接打开即可

## 📱 手机使用

1. 在微信中点击网址链接
2. 直接在微信中使用，或复制网址到浏览器
3. 建议添加到手机桌面
